## Important Implementation Rule

Use these projects as inspiration only:

- G-Retriever: https://github.com/xiaoxinhe/g-retriever
- General-Reasoner: https://github.com/TIGER-AI-Lab/General-Reasoner

Do NOT directly implement their CUDA/GPU-heavy training workflows in Stage 1.

G-Retriever is a research framework for textual graph QA, and General-Reasoner is a training/reasoning framework, so for this Mac/local MVP, implement a lightweight GraphRAG version using LangChain, NetworkX, Chroma, and Qwen. 

G-Retriever’s repo describes it as a retrieval-augmented generation framework for textual graph understanding and QA, so using it as architectural inspiration is appropriate. General-Reasoner is described as a reasoning training framework across domains, so it should not be treated as a simple app dependency for this MVP.
